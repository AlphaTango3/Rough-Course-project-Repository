# Getting and Cleaning Data Project

## Files
- run_analysis.R
- tidy_data.txt
- CodeBook.md

## What the script does
1. Merges train + test
2. Extracts mean/std variables
3. Applies activity labels
4. Cleans variable names
5. Creates grouped average dataset

## How to run
Place dataset in working directory and run:
source("run_analysis.R")
